Every hookMọi hook

47 hooks

A hook is a small program the harness runs around a tool call. Its tier is a code constant (HOOK_CLASS) — config can toggle a hook on/off but never change its tier, and the fail posture follows the tier. See the rig for how the gate uses them.Hook là một chương trình nhỏ harness chạy quanh mỗi lần gọi tool. Hạng của nó là hằng số trong code (HOOK_CLASS) — config bật/tắt được nhưng không đổi được hạng, và tư thế-khi-lỗi theo hạng. Xem the rig để biết gate dùng chúng thế nào.
